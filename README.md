# YSQ-S3 — Опитувальник ранніх дезадаптивних схем Янга

Веб-версія опитувальника YSQ-S3 (90 тверджень, 18 схем) з підрахунком сум, середніх та рівнів по кожній схемі. Статичний сайт без бекенду — відкривається подвійним кліком або розгортається на GitHub Pages.

## Можливості

- 90 запитань, шкала 1–6, автопрокрутка до наступного запитання
- Введення імені перед початком тесту
- Прогрес-бар; кнопка «Залишилось N» прокручує до першого неотриманого запитання
- Результат: сума, середнє та рівень по кожній з 18 схем, загальний бал
- Акордеон з розбивкою «запитання → бал» по кожній схемі
- Збереження прогресу, відповідей і поточного екрана в `sessionStorage` (переживає оновлення сторінки)
- Друк / збереження у PDF («Зберегти / Друк»): акордеони розкриваються, службові блоки та кнопки приховуються, орієнтація — книжкова

## Структура

```
index.html         розмітка, підключення, <template>-шаблони
css/styles.css     стилі
js/app.js          логіка
data/questions.js  90 запитань (const QUESTIONS)
data/schemas.js    схеми, шкала, пороги рівнів (як рахувати)
documents/         вихідні матеріали методики (PDF, scoring)
```

## Запуск локально

Подвійний клік по `index.html` — працює одразу, без сервера.

Якщо потрібно як на проді (через `http://`) — використовуйте Live Server у VS Code або:

```
python3 -m http.server
```

і відкрийте `http://localhost:8000/`.

## Розгортання на GitHub Pages

1. Запуште репозиторій на GitHub (головний файл уже називається `index.html`, тож відкриватиметься за кореневою адресою).
2. Settings → Pages → Source: гілка `main`, тека `/ (root)`.
3. Сайт буде доступний за адресою `https://<username>.github.io/<repo>/`.

## Як редагувати

- **Запитання** — `data/questions.js`, масив `QUESTIONS`. Кількість запитань усюди рахується автоматично з `QUESTIONS.length`.
- **Схеми та підрахунок** — `data/schemas.js`:
  - `SCHEMAS` — які запитання входять у кожну схему (`items` — номери запитань, від 1);
  - `SCALE_LABELS` — підписи до балів шкали;
  - `SCALE_MAX` — максимум шкали;
  - `LEVELS` — пороги рівнів за середнім балом.
- **Розмітка динамічних блоків** — `<template>` у `index.html` (`tpl-question`, `tpl-opt`, `tpl-schema`, `tpl-qrow`); `js/app.js` клонує їх і підставляє значення.

## Технічні нотатки

- Дані лежать у `.js` (а не `.json`), бо при відкритті файлу подвійним кліком (`file://`) браузер блокує `fetch()` локальних JSON. Файли-дані підключаються через `<script src>`, що працює і локально, і на GitHub Pages.
- Стан зберігається в `sessionStorage` — очищається при закритті вкладки та при натисканні «На початок» / «Пройти знову».
